# Fábrica de Monstros – Pensamento Computacional

Este projeto implementa um algoritmo baseado nas regras apresentadas nos slides da disciplina.

## Como funciona

A classificação do monstro é feita assim:
- A cabeça determina a primeira parte do nome
- A predominância das características determina a segunda parte
- Se houver empate, a família dos olhos decide
- Se o monstro não possuir características, a categoria vira “nulo”

## Arquivos
- monstro.c – código em C
- pseudocodigo.txt – versão em pseudocódigo
- Exemplo de execução está no chat do desenvolvimento

## Como compilar
gcc monstro.c -o monstro

## Como executar
./monstro
